# Welcome to Aruna

**Turn Invoices Into Public Goods Funding**

Aruna is a DeFi protocol that transforms invoice financing into sustainable public goods funding. Every dollar deposited earns competitive yields while automatically donating 25% to Ethereum ecosystem projects.

## What You'll Learn

This documentation will help you understand:

- **The Problem** - Why public goods need sustainable funding
- **The Solution** - How Aruna creates recurring donations automatically
- **Smart Contracts** - Deployed addresses and verification
- **How It Works** - Step-by-step user flows
- **Key Concepts** - Understanding the 70/25/5 model
- **Use Cases** - Real-world examples for businesses, investors, and public goods

## Quick Start

**For Investors:**
1. Connect your wallet to [Aruna App](https://aruna.finance)
2. Deposit USDC to Aave or Morpho vault
3. Earn 5.46% effective APY
4. Automatically fund public goods with 25% of your yield

**For Businesses:**
1. Submit invoice commitment
2. Lock 10% collateral, receive 3% instant grant
3. Build on-chain reputation
4. Settle invoice to unlock collateral

## The Innovation

Traditional donation models require repeated user actions. Aruna makes public goods funding automatic:

- **One deposit** → Perpetual funding
- **No governance** → Immutable 25% allocation
- **Full transparency** → On-chain verification
- **Zero friction** → No repeated decisions
